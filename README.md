# Hi, I'm Ethan Meredith

UK-based developer with a background in IT support and systems administration,
building toward a full-time role in software development.

By day I manage users, workflows, and infrastructure using Azure AD and the
Microsoft Power Suite. Outside of that I build things; desktop apps, web tools,
and whatever problems feel worth solving.

## What I'm working on

**[Jackdaw](https://github.com/MeredithEth0/Jackdaw)**: a free,
self-contained Windows desktop app for downloading and converting video/audio
from 1000+ sites. Built with Tauri, yt-dlp, and ffmpeg.
